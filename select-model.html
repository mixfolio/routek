Выборочное лазерное спекание (SLS)

SLS используется как для изготовления прототипов, так и для мелкосерийного производства функциональных пластиковых деталей с высокими механическими свойствами.

Многоструйное моделирование (MJF)

Многоструйное моделирование (MJF) от HP можно использовать как для создания прототипов, так и для производства мелкосерийных функциональных пластиковых деталей.

Рассчитать заказ
Прямое лазерное спекание металла (DMLS)

Метод DMLS позволяет производить высококачественные детали из металла для применения в аэрокосмической, автомобильной и медицинской промышленности

Рассчитать заказ
Моделирование методом наплавления (FDM)

FDM - это наиболее широко доступный процесс 3D-печати, в основном используемый для создания недорогих прототипов и тестирования дизайна в очень короткие сроки.

Рассчитать заказ
Стереолитография (SLA)

SLA наиболее подходит для визуальных применений, где требуются литьевые формы, гладкая поверхность и высокий уровень детализации деталей.

Рассчитать заказ
Струйное нанесение нескольких материалов (Polyjet)

Polyjet производит детали высочайшей точности размеров с очень гладкой поверхностью, используемые для визуальных прототипов.
